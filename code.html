The Code for FizzBuzz

                    
                        //get values from the user. We need to get the fizz and the buzz value.
                        function getValues(){
                        
                            //get the user values from the page
                            let fizzValue = document.getElementById("fizzValue").value;
                            let buzzValue = document.getElementById("buzzValue").value;
                        
                            //parse for numbers
                            fizzValue = parseInt(fizzValue);
                            buzzValue = parseInt(buzzValue);
                        
                            //check that the numbers are integers
                            if(Number.isInteger(fizzValue) && Number.isInteger(buzzValue)){
                        
                            //call fizzBuzz() function
                            let fbArray = fizzBuzz(fizzValue,buzzValue);
                        
                            //call displayData() function and write the values to the screen
                            displayData(fbArray);
                        
                            } else {
                                alert("You must enter integers")
                            }
                        }
                        
                        //do fizz buzz
                        function fizzBuzz(fizzValue, buzzValue){
                        
                            //initialize the returnArray[]
                            let returnArray = [];
                        
                            //loop from 1 to 100
                            for (let i = 1; i <= 100; i++) {
                        
                                //check if divisible by 3 and 5
                                //if so push `FizzBuzz` into the array and not the number
                                if(i % fizzValue == 0 && i % buzzValue == 0){
                                    returnArray.push('FizzBuzz');
                        
                                //check to see if divisble by fizz value 3
                                //if so push `Fizz` into the array and not the number
                                } else if(i % fizzValue == 0) {
                                    returnArray.push('Fizz');
                                    
                                //check to see if divisble by buzz value 5
                                //if so push `Buzz` into the array and not the number
                                } else if(i % buzzValue == 0) {
                                    returnArray.push('Buzz');
                                
                                //If it isn't divisible by either number push the number into the array
                                } else {
                                    returnArray.push(i);  
                                }        
                            }
                            return returnArray;
                        }
                        
                        //loop over the array and create a table row for each item
                        function displayData(fbArray){
                        
                            //get the table body element from the page
                            let tableBody = document.getElementById("results");
                        
                            //get the template row
                            let templateRow = document.getElementById("fbTemplate");
                        
                            //clear table first
                            tableBody.innerHTML = "";
                        
                            for (let index = 0; index < fbArray.length; index+=5) {
                                
                                let tableRow = document.importNode(templateRow.content, true);
                        
                                //grab just the tds and put them into array
                                let rowCols = tableRow.querySelectorAll("td");
                                              
                                rowCols[0].textContent = fbArray[index];                                    
                                rowCols[1].textContent = fbArray[index+1];                          
                                rowCols[2].textContent = fbArray[index+2];                             
                                rowCols[3].textContent = fbArray[index+3];                                     
                                rowCols[4].textContent = fbArray[index+4];
                                
                                tableBody.appendChild(tableRow);
                            }  
                        }

FizzBuzz is an application that consists of three functions working together.

getValues()

Inside this function it declares two variables and assigns the input from the user to those variables. In order to do some input validation it attempts to parse the input with a parseInt and then followed up by an if statement that checks if the user input are indeed numbers. If the input passes the input validation then this function has an array that passes the user input into the fizzBuzz() function and then passes the results of that function into another function called displayData(). Lastly if the user input did not pass the if statement check the if statement just alerts the user that they must enter integers.

fizzBuzz()

The fizzBuzz() function is used to process user input and perform some logic on that user input. Right off the bat it initializes an empty array to be used later in the function. Since we want to iterate to 100 a for loop is declared that checks both the fizzValue and buzzValue for various parameters and based on the results pushes either Fizz, Buzz, FizzBuzz or the number to the array that was declared earlier in this case it is called returnArray[]. Eventually the for loop will conclude iterating over the numbers and then the return statement will return the array.

displayData()

This function is used to take the processed user data from the fizzBuzz() and display it on the page. It takes an array as a parameter and declares two variables that take various elements from the page. For example the tableBody.innerHTML = ""; statement ensures that tableBody is empty. Using a for loop we begin to iterate over the array that was passed into it and begin displaying it by appending the information via tableBody.appendChild(tableRow);
